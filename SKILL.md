# Paper Reading Skill

## 目标

对用户提供的论文 PDF 进行结构化阅读，帮助用户快速判断论文价值、理解方法主线，并从复现视角识别关键细节、限制和改进方向。

## 工作流程

### Pass 1：快速判断

回答 5C：

1. Category：论文属于什么研究方向或任务类型。
2. Context：论文试图解决什么背景问题。
3. Correctness：论证、实验和结论是否看起来可靠。
4. Contributions：核心贡献是什么。
5. Clarity：论文表达是否清晰，哪些部分需要进一步核查。

### Pass 2：技术主线

梳理论文的方法流程，重点说明：

1. 系统输入是什么。
2. 系统输出是什么。
3. 方法由哪些模块或步骤组成。
4. 训练、推理或实验设置是什么。
5. 主要实验结果和对比结论是什么。

### Pass 3：复现与批判

从复现角度分析：

1. 关键假设和前置条件。
2. 必须复现的核心模块、数据、指标和实验。
3. 论文的创新点。
4. 可能的局限性。
5. 后续可改进方向。

## 输出要求

输出 Markdown 报告。能识别章节、图号、表号或页码时应标注；无法可靠识别的内容必须明确说明，不得编造论文中不存在的信息。
